# Manos Neofotistos Portfolio

A minimal static portfolio built with Next.js, TypeScript, Tailwind CSS, and the App Router.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Edit content

Most personal content lives in `data/site.ts`. Replace the placeholder social links, project URLs, copy, tags, and experience entries there.

The main image is `public/profile.jpg`. Replace it with your own image using the same filename, or update `site.hero.image.src` in `data/site.ts`.
